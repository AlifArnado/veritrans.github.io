VT-Link

The tutorial below will assist you in integrating VT-Link. If further assistance is required, please contact us at support@veritrans.co.id or visit our support page.

VT-Link facilitates a payment feature for your online store by redirecting the customer to Veritrans payment page through a link.

Overview


Required Fields

The following fields are required for Link Creation:

  1. Order ID
  2. Item Name
  3. Item Price


If the transaction is appointed for a specific customer, VT-Link will be sent to the specified customer email.
These additional fields are required:

  1. Customer Email
  2. First Name
  3. Last Name
  4. Phone
Warning: Note: Expiration date is the last day when VT-Link can be used.


Processing Payment

Veritrans will generate a link that redirects the user to Veritrans payment page to complete transaction.
All payment methods available for the merchant will be accesable at the payment page.
After the transaction is performed, user will be redirected to the URL managable through MAP Configuration.

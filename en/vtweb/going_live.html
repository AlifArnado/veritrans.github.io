Going Live with VT-Web

The tutorial below will assist you in changing the configuration from Sandbox to Production. Please choose the following integration method. If further assistance is required, please contact us at support@veritrans.co.id or visit our support page.


  • PHP
  • Magento
  • Opencart
  • Prestashop
  • Woocommerce


If you have completed integrating in Sandbox (Testing Environment) and ready to go live in Production Environment, there are a few steps that need to be made:

  • Contact Veritrans Support to migrate from Sandbox to Production Environment.
    Once confirmed, you can now login to Veritrans Production Dashboard at https://my.veritrans.co.id/login with the current credentials.
  • Set Veritrans-PHP library to Production Environment.
    This can be done by changing the value Veritrans_Config::$isProduction into true. 
<?php
include_once('Veritrans.php');

// Set our production server key
Veritrans_Config::$serverKey = '<your production server key>';

// Use production account
Veritrans_Config:$isProduction = true;
...

?>


You are now integrated live with Veritrans service and ready to start transaction!

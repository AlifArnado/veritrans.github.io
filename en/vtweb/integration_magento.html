VT-Web Magento Integration Steps

  1. Plugin Installation
  2. Transaction Test


  • Step 1Plugin Installation
  • Step 2Transaction Test

Requirements

  1. An online store with Magento infrastructure. This plugin is tested with Magento version 1.9.
  2. PHP minimum version 5.4.
  3. Veritrans plugin for Magento [ Github | Zip ].

Installation Process

  1. Extract VTWebmagento-master.zip file you have previously downloaded.
  2. Upload / copy app, lib, dan skin folder that you have extracted into your Magento directory.

Plugin Configuration

In order to configure Veritrans-Magento plug-in:

  1. Access your Magento admin page.
  2. Go to System - Configuration menu in Magento admin page.
  3. In Sales-Payment Methods option, find the Veritrans tab. If Veritrans tab is not present, click Save Config beforehand.
  4. Click general to start configuration.
  5. Insert required fields following the figure below and click Save Config.



  6. Login to MAP portal to configure a few options below:

    URL Role Redirect URL
    Payment Notification URL http://[your_shop’s_homepage]/vtweb/payment/notification
    Finish Redirect URL http://[your_shop’s_homepage]/vtweb/payment/response
    Error Redirect URL http://[your_shop’s_homepage]/vtweb/payment/response
    Unfinish Redirect URL http://[your_shop’s_homepage]/vtweb/payment/response


  1. Perform a successful transaction in your online store by inputing a dummy credit card number as followed (Sandbox Mode):

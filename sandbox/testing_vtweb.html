Testing VT-Web on Sandbox Environment

If request charge with payment_type “vtweb” is done with configuration (merchant_id, server_key, client_key, dan URL endpoint) mode of Sandbox, Veritrans will respond back with “redirect_url” address to VT-Web page on Sandbox Environment. VT-Web on Sandbox mode is marked with “Testing mode” sign on the top right hand corner. On VT-Web mode of Sandbox, you will be able to conduct trial transaction using a special credit card number dedicated only for use on Sandbox mode.

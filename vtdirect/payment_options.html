Payment Options

Currently Veritrans provide 2 payment options:

  1. Mandiri ClickPay
  2. CIMB Clicks

Here is a tutorial for integrate using payment options to suit your needs.

Mandiri ClickPay

Charge Transaction

To perform a charge transaction using Mandiri ClickPay, several steps must be carried out as follows:

  1. "payment_type" must be set into"mandiri_clickpay"
  2. JSON new object must be added, with name "mandiri_clickpay",and has pairing name/value as follows:
    • "card_number" → credit card number
    • "input1" → last 10 digit from credit card number
    • "input2" → sum of total transactions
    • "input3" → merchant order id
    • "token" → token that have been filled by customer
{
  "payment_type": "mandiri_clickpay",
  "transaction_details": {
    "order_id": "201404031521",
    "gross_amount": 10000
  },
  "item_details": [
    {
      "id": "1388998298204",
      "price": 1000,
      "quantity": 10,
      "name": "Mie Ayam Komplit"
    }
  ],
  "customer_details": {
    "email": "jcustomer@example.com",
    "first_name": "John",
    "last_name": "Customer",
    "phone": "+6281 1234 1234"
  },
  "mandiri_clickpay": {
    "card_number": "4111111111111111",
    "input1": "1111111111",
    "input2": "10000",
    "input3": "00000",
    "token": "000000"
  }
}

Response

Sample response that obtained from charge transaction for Mandiri Clickpay as follows:

{
    "transaction_id": "cddd6c1f-3b03-4c89-b0a8-0ea7dccf5217",
    "order_id": "201404031721",
    "gross_amount": "10000",
    "payment_type": "mandiri_clickpay",
    "transaction_time": "2014-04-03 17:21:46",
    "transaction_status": "settlement",
    "status_code": "200",
    "status_message": "OK, Mandiri Clickpay transaction is successful"
}

CIMB Clicks

Charge Transaction

To perform charge transaction using CIMB Clicks, several steps must be carried out as follows:

  1. "payment_type" must be set into "cimb_clicks"
  2. JSON new object must be added, with name "cimb_clicks", and has pairing name/value as follows:
  • "description" → transaction description
 {
  "payment_type": "cimb_clicks",
  "transaction_details": {
    "order_id": "201404141421",
    "gross_amount": 10000
  },
  "item_details": [
    {
      "id": "1388998548953",
      "price": 1000,
      "quantity": 10,
      "name": "Mie Ayam Komplit"
    }
  ],
  "customer_details": {
    "email": "jcustomer@example.com",
    "first_name": "John",
    "last_name": "Customer",
    "phone": "+6281 1234 1234"
  },
  "cimb_clicks": {
    "description": "Contoh Deskripsi"
  }
}

Response

Sample response that obtained from charge transaction for CIMB Clicks as follows: 

{
  "status_code": "201",
  "status_message": "OK, CIMB Clicks transaction is successful",
  "transaction_id": "ff05337c-6c94-4f70-8e81-35acd89b688e",
  "order_id": "201404141421",
  "payment_type": "cimb_clicks",
  "transaction_time": "2014-04-14 16:03:51",
  "transaction_status": "pending",
  "redirect_url": "https://api.sandbox.veritrans.co.id/cimb-clicks/request?id=ff05337c-6c94-4f70-8e81-35acd89b688e",
  "gross_amount": "10000.00"
}

Other Command

The following tutorial will help you get to know the commands that can be used in the VT-Direct. If further assistance is required, please contact us at support@veritrans.co.id or visit our support page.

In the integration process, we have explained how to charge a transaction using VT-Direct. In addition to the charge operation, there are other operations related to the transaction you can do:

  1. Status transaction
  2. Approve transaction
  3. Cancel transaction
  • Status Transaksi
  • Approve Transaksi
  • Cancel Transaksi

Status transaction

In certain situations, you can see the status of the transaction using the function Veritrans_Transaction::status. You simply give the order ID when using this function.

<?php
include_once('Veritrans.php');

Veritrans_Config::$serverKey = "< Your Server Key >";
Veritrans_Config::$isProduction = false;

$order_id = "<your order id>";

$result = Veritrans_Transaction::status($order_id);
?>

Structure and instance variables $result are as follows:

array(
    "transaction_id" => "05dfbd23-b767-4748-8fe9-7e30769e1857",
    "order_id" => "order53",
    "gross_amount" => "100000.00",
    "payment_type" => "credit_card",
    "challenge [masked_card]" => "411111-1111",
    "transaction_time" => "2014-03-27 17:25:12",
    "transaction_status" => "capture",
    "status_code" => "200",
    "signature_key" => "0b063c979a872640b81b1aef2bb4f093388bc6e6b91b164beda58da1b102de277791074dcc10f237f4583864cfa0691799e906ef3ca50ff34b8ecd198e1fa2e4",
    "status_message" => "OK, Credit Card transaction is successful"

Approve transaction

You can approve the transaction using the function Veritrans_Transaction::approve (). Approve only be performed on the transaction who have challenge status from Fraud Detection System (FDS). You simply give the order ID when using this function.

<?php
include_once('Veritrans.php');

Veritrans_Config::$serverKey = "< Your Server Key >";
Veritrans_Config::$isProduction = false;

$order_id = "<your order id>";

$result = Veritrans_Transaction::approve($order_id);
?>

Structure and instance variables $result are as follows:

array( 
    "status_code" => "200",
    "status_message" => "OK, transaction has been approved",
    "transaction_id" => "a9095770-4e7a-4880-9961-11b45d42359b",
    "order_id" => "825524557",
    "payment_type" => "credit_card",
    "transaction_time" => "2014-08-15 19:16:55",
    "transaction_status" => "capture",
    "fraud_status" => "accept",
    "masked_card" => "481111-1114",

    "gross_amount" =>"100000.00"
);

Cancel transaction

You can do cancel transaction in VT-Direct using Veritrans_Transaction::cancel() function. Cancel the transaction can only be done on a transaction that has not settlement yet. You simply give the order ID when using this function.

<?php
include_once('Veritrans.php');

Veritrans_Config::$serverKey = "< Your Server Key >";
Veritrans_Config::$isProduction = false;

$order_id =  "<your order id>";

$result = Veritrans_Transaction::cancel($order_id);
?>

Structure and instance variables $result are as follows:

array(
    "status_code" =>"200",

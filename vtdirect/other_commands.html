Get transaction status

Description Get status information from transaction with particular id(transaction_id or order_id).
Endpoint Sandbox : https://api.sandbox.veritrans.co.id/v2/{id}/status
Production : https://api.veritrans.co.id/v2/{id}/status
HTTP Method GET
HTTP Header Accept: application/json
Content-Type: application/json
Authentication: Basic base64(server_key:)
Body Request blank

Approve transaction

Description Approve transaction with particular id(transaction_id or order_id) that have challenge status from 'Fraud Detection System.
Endpoint Sandbox : https://api.sandbox.veritrans.co.id/v2/{id}/approve
Production : https://api.veritrans.co.id/v2/{id}/approve
HTTP Method POST
HTTP Header Accept: application/json
Content-Type: application/json
Authentication: Basic base64(server_key:)
Body Request blank

Cancel transaction

Description Cancel transaction with particular(transaction_id or order_id). Cancellation can only be done if that transaction has not been through the settlement process.
Endpoint Sandbox : https://api.sandbox.veritrans.co.id/v2/{id}/cancel
Production : https://api.veritrans.co.id/v2/{id}/cancel
HTTP Method POST
HTTP Header Accept: application/json
Content-Type: application/json
Authentication: Basic base64(server_key:)
Body Request blank

Transcript transaction

Description Get detail information log from a transaction from particular ID(transaction_id or order_id).
Endpoint Sandbox : https://api.sandbox.veritrans.co.id/v2/{id}/transcript
Production : https://api.veritrans.co.id/v2/{id}/transcript
HTTP Method GET
HTTP Header Accept: application/json
Content-Type: application/json
Authentication: Basic base64(server_key:)
Body Request blank

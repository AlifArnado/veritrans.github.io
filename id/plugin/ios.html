Integrasi iOS

Dokumentasi pada halaman ini bertujuan untuk membantu anda melakukan integrasi dengan metode pembayaran kartu kredit pada aplikasi iOS menggunakan VT-Direct. Apabila butuh bantuan, silahkan hubungi kami di support@veritrans.co.id atau kunjungi halaman support kami.


Perkenalan

VTiOSLib adalah library iOS yang digunakan untuk melakukan proses mendapatkan token dari Veritrans dengan menggunakan VT-Direct. Dengan mempergunakan library ini, anda bisa mendapatkan data kartu kredit secara aman tanpa harus menyimpan data tersebut di server milik anda, library ini juga sudah mendukung proses mendapatkan token 3D Secure.


Note: Untuk proses pengembangan dan testing, anda harus memiliki akun sandbox Veritrans.


Alur Proses Transaksi

Ada tiga pihak yang terlibat dalam proses pembayaran untuk VT-Direct: anda, pelanggan anda, dan Veritrans. Berikut adalah penjelasan bagaimana VT-Direct bekerja.

  1. Klik Pay: Pelanggan anda memasukan detail kartu kredit di aplikasi iOS anda, dan menekan tombol PAY untuk menyelesaikan pembayaran.

  2. Mengirim data: ketika pelanggan anda menekan tombol PAY, library iOS akan menggabungkan informasi kartu kredit yang sensitif dengan client key. Gabungan data tersebut dikirimkan ke server Veritrans untuk ditukar dengan onetime-token ("Token").

  3. Mendapatkan Token: Veritrans akan mengembalikan Token ke aplikasi iOS anda. Token akan valid selama 3 menit pada transaksi kartu kredit biasa dan 10 menit jika fitur 3D Secure digunakan. Token tidak bisa digunakan lagi jika melewati durasi waktu tersebut.

  4. Mengirim Token: Token kemudian dikirimkan ke server anda. Harap di perhatikan, anda hanya akan menerima Token sebagai pengganti detail dari kartu kredit. Anda tidak akan menyimpan ataupun memproses informasi dari kartu kredit. Karena hal tersebut, anda tidak perlu berurusan dengan PCI DSS compliance.

  5. Request Charge: Anda melakukan charge transaksi ke Veritrans menggunakan Token ditambah beberapa informasi lainnya (contoh: detail pelanggan, detail produk, alamat).

  6. Transaction Response Handling: Veritrans mengembalikan respon status transaksi, seperti

    • Capture
    • Settlement

  7. Memberikan informasi transaksi: Anda memberitahukan hasil transaksi kepada pelanggan anda.

STEP 1: INSTALASI

Ada dua langkah yang harus dilakukan agar aplikasi anda dapat menggunakan library iOS Veritrans.

  1. Import VTiOSAPI.a

    • Download versi terbaru VTiOSAPI.a dari github.
    • Membuat folder Lib dibawah folder root project anda.
    • Salin VTiOSAPI.a ke dalam folder Lib yang sudah dibuat.
    • Klik project anda dari Project Structure di XCode.
    • Pilih Build Phases
    • Dibawah Link Binary with Libraries klik simbol + dan pilih add other
    • Cari folder Lib yang tadi sudah dibuat dan pilih VTiOSAPI.a untuk dimasukan oleh XCode kedalam struktur project anda.

  2. Membuat file Bridging-Header

    • Salin folder include yang terdapat pada file ZIP yang sudah di download ke folder root project anda.
    • Pindahkan folder include dari finder ke XCode Project Structure.
    • Klik project Structure dan pilih tab Build Settings.
    • Cari Objective-C Bridging header dibawah Swift Compiler - Code Generation.
    • Double Klik field input dan masukan $SOURCE_ROOT/include/VTiOSAPI-Bridging-Header.h (pastikan folder include ada di dalam root project anda).
    • Saat ini VTiOSAPI dapat digunakan untuk project anda.


    Note: Semua file yang dibutuhkan ada di dalam folder ZIP.


STEP 2: Mendapatkan Token

  • Set VTConfig ke mode Sandbox dan memasukan Client Key.
//set environment
VTConfig.setVT_IsProduction(false)

//set client key
VTConfig.setCLIENT_KEY("< Your Client Key >”);


  • Membuat objek VTDirect serta mengatur rincian data kartu untuk mendapatkan token.
//Create VTDirect Object. Its config is Based on VTConfig Class that you set previously
var vtDirect = VTDirect()

//Create VTCardDetails Object to be set to VTDirect Object
var cardDetails = VTCardDetails()

//TODO: Set your card details based on user input.
//this is a sample
cardDetails.card_number = "4811111111111114"
cardDetails.card_cvv = "123"
cardDetails.card_exp_month = 1
cardDetails.card_exp_year = 2020;

//set true or false to enable or disable 3dsecure
cardDetails.secure = true
cardDetails.gross_amount = "100000"

//Set VTCardDetails to VTDirect
vtDirect.card_details = cardDetails


  • Meminta Token
//Simply Call getToken function and put your callback to handle data
vtDirect.getToken { (token:VTToken!, ex : NSException!) -> Void in
            if(ex == nil){

                //use token anyhow you want, maybe send it to your server. Example here to check whether you are using 3dsecure feature or not.
                if(token.redirect_url != nil){
                    
                    //using 3d secure
                    let webView:UIWebView = UIWebView(frame: CGRectMake(0, 0, 400, 420))
                    webView.loadRequest(NSURLRequest(URL: NSURL(string:token.redirect_url)!))
                    webView.delegate = self
                    self.view.addSubview(webView)
                    
                }
                
                }else{
                    //Something is wrong, get details message by print ex.reason
                    println(ex.reason)
            }
        }


STEP 3: Mengirim token ke server

Setelah mendapatkan token, anda harus mengirimkan token tersebut ke server agar dapat melakukan proses charge.

var request = NSMutableURLRequest(URL: NSURL(string: "http://128.199.141.15:9091/index.php")!)
var session = NSURLSession.sharedSession()
request.HTTPMethod = "POST"

//put your token on a variable to be retrieved again in this part
var bodyData = "token-id=\(self.token!.token_id)&price=\(totalPrice)"
request.HTTPBody = bodyData.dataUsingEncoding(NSUTF8StringEncoding)
request.addValue("application/json", forHTTPHeaderField: "Accept")

//Asynchronously get data from your server
var task = session.dataTaskWithRequest(request, completionHandler: { (data, response, error) -> Void in
                if(error == nil){
             //we’ve got response from server, print it here
                    var strData = NSString(data: data, encoding: NSUTF8StringEncoding)
                    println("Body: \(strData!)")                    
                    
                }else{
              //something is wrong, print localizedDescription
                    println("Error: \(error.localizedDescription)")
                }
                
            })
            task.resume()


STEP 4: Melakukan Proses Charge

Setelah mengirimkan token ke server, sekarang saatnya melakukan proses charge dari server merchant ke server Veritrans dengan menggunakan server key. Contoh berikut menggunakan PHP sebagai processing server dengan menggunakan library PHP milik Veritrabs.

File : index.php

<?php
include_once("Veritrans.php");

if(empty($_POST['token-id'])){
        echo "Empty token-id";
        exit;
}

Veritrans_Config::$serverKey = "< Your server key >";
Veritrans_Config::$isProduction = false;

$token_id = $_POST['token-id'];
//echo $token_id;

$transaction_details = array(
  'order_id'    => rand(),
  'gross_amount'  => $_POST['price']
);

$transaction_data = array(
  'payment_type'      => 'credit_card',
  'credit_card'       => array(
    'token_id'  => $token_id,
    'bank'    => 'bni'
    ),
  'transaction_details'   => $transaction_details,
  );
try{
        $result = Veritrans_VtDirect::charge($transaction_data);
        $json = json_encode($result);
        echo "{\"status\":\"success\", \"body\" : $json}";
}catch(Exception $e){
        echo "{\"status\":\"error\",\"body\": \".$e.\"}";
}

?>

Bila anda menggunakan bahasa pemrograman lain, silahkan cek library kami di sini.

Anda dapat melihat contoh toko sederhana dari aplikasi yang menggunakan library kami di sini.

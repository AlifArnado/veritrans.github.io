Perintah-Perintah Lain

Tutorial berikut akan membantu anda mengenal perintah-perintah yang dapat digunakan pada VT-Direct. Apabila butuh bantuan, silahkan hubungi kami di support@veritrans.co.id atau kunjungi halaman support kami.


Pada proses integrasi, kami telah menjelaskan cara charge transaksi menggunakan VT-Direct. Di samping operasi charge, ada operasi lain terkait transaksi yang dapat anda lakukan:

  1. Status transaksi
  2. Approve transaksi
  3. Cancel transaksi


  • Status Transaksi
  • Approve Transaksi
  • Cancel Transaksi


Status transaksi

Pada situasi tertentu, anda dapat melihat status transaksi menggunakan fungsi Veritrans_Transaction::status. Anda cukup memberikan order ID saat menggunakan fungsi ini. 

<?php
include_once('Veritrans.php');

Veritrans_Config::$serverKey = "< Your Server Key >";
Veritrans_Config::$isProduction = false;

$order_id = "<your order id>";

$result = Veritrans_Transaction::status($order_id);
?>

Approve transaksi

Anda dapat melakukan approve transaksi menggunakan fungsi Veritrans_Transaction::approve(). Approve hanya dapat dilakukan pada transaksi yang memperoleh status challenge dari Fraud Detection System (FDS). Anda cukup memberikan order ID pada saat menggunakan fungsi ini. 

<?php
include_once('Veritrans.php');

Veritrans_Config::$serverKey = "< Your Server Key >";
Veritrans_Config::$isProduction = false;

$order_id = "<your order id>";

$result = Veritrans_Transaction::approve($order_id);
?>


Struktur dan contoh variabel $result ketika operasi approve berhasil adalah sebagai berikut: 

array( 
	"status_code" => "200",
	"status_message" => "Success, transaction is approved",
	"transaction_id" => "2af158d4-b82e-46ac-808b-be19aaa96ce3",
	"masked_card" => "451111-1117",
	"order_id" => "2014112112",
	"payment_type" => "credit_card",
	"transaction_time" => "2014-11-27 10:05:10",
	"transaction_status" => "capture",
	"fraud_status" => "accept",
	"approval_code" => "1417057511311",
	"bank" => "bni",

Cancel transaksi

Anda dapat melakukan perintah cancel transaksi pada VT-Direct menggunakan fungsi Veritrans_Transaction::cancel(). Cancel transaksi hanya dapat dilakukan pada transaksi yang belum melewati status settlement. Anda cukup memberikan order id sebagai parameter dari fungsi ini. 

<?php
include_once('Veritrans.php');

Veritrans_Config::$serverKey = "< Your Server Key >";
Veritrans_Config::$isProduction = false;

$order_id =  "<your order id>";

$result = Veritrans_Transaction::cancel($order_id);
?>


Struktur dan contoh variabel $result ketika operasi cancel berhasil adalah sebagai berikut: 

array(
	"status_code" => "200",
	"status_message" => "Success, transaction is canceled",
	"transaction_id" => "2af158d4-b82e-46ac-808b-be19aaa96ce3",
	"masked_card" => "451111-1117",
	"order_id" => "2014112112",
	"payment_type" => "credit_card",
	"transaction_time" => "2014-11-27 10:05:10",
	"transaction_status" => "cancel",
	"fraud_status" => "accept",
	"bank" => "bni",
	"gross_amount" => "10000.00"
);

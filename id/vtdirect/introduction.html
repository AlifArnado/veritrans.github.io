Perkenalan VT-Direct

VT-Direct adalah RESTful API yang memungkinkan merchant untuk menggunakan sistem pembayaran Veritrans untuk memproses pembayaran langsung di website merchant. VT-Direct memungkinkan pembeli untuk menyelesaikan proses pembayaran di website merchant tanpa perlu dialihkan (di-redirect) ke halaman Veritrans, tetapi dengan tetap meminimalisir risiko keamanan yang muncul pada saat pemrosesan pembayaran menggunakan kartu kredit.

VT-Direct juga dilengkapi dengan script Javascript yang dipasang di website merchant untuk mengirim data kartu kredit secara langsung ke server Veritrans tanpa terlebih dahulu disimpan di server merchant.

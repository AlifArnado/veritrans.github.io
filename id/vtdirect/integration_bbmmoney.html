Integrasi VT-Direct (BBM Money)

Setiap metode pembayaran memiliki tahapan integrasi yang berbeda. Dokumentasi pada halaman ini bertujuan untuk membantu anda melakukan integrasi VT-Direct dengan metode pembayaran BBM Money. Apabila butuh bantuan, silahkan hubungi kami di support@veritrans.co.id atau kunjungi halaman support kami.



Salah satu metode pembayaran yang di tawarkan oleh Veritrans adalah BBM Money. Dengan menggunakan layanan ini, pelanggan anda akan memiliki pilihan untuk melakukan pembayaran melalui BBM Money, baik ke akun BBM Money lainnya atau ke rekening virtual account Bank Permata.

Untuk mengaktifkan metode pembayaran BBM Money (otomatis aktif pada mode Sandbox), anda perlu menghubungi sales representative Veritrans.

Info: Note:
- silahkan menuju Referensi API untuk informasi lebih lengkap tentang atribut BBM Money.

Tahapan Integrasi VT-Direct BBM Money

  1. Membuat Form
  2. Charge Transaksi
  3. Membuat Link BBM Money
  4. Status Transaksi

Berbeda dengan metode pembayaran lain (kartu kredit, Mandiri Clickpay dan CIMB Clicks), proses utama pembayaran BBM Money tidak dilakukan di halaman anda melainkan melalui ponsel yang sudah registrasi dengan BBM Money.

  • Tahap 1Membuat Form
  • Tahap 2Charge Transaksi
  • Tahap 3Membuat Link BBM Money
  • Tahap 4Status Transaksi


Membuat Form

Form pembayaran BBM Money akan kita buat di file form_vtdirect_bbmmoney.html. Karena sebagian besar proses pembayaran tidak dilakukan di halaman merchant, form yang akan kita buat hanya berisi petunjuk pembayaran dan tombol bayar menggunakan BBM Money. Ketika pelanggan anda menekan tombol ini, proses charge transaksi dilakukan pada tahap selanjutnya.

Berikut adalah contoh form pada file form_vtdirect_bbmmoney.html: 

<head>
	<title>BBM Money</title>
</head>
<body>
	<h1>BBM Money</h1>
	<b>Saya akan melakukan pembayaran dengan metode BBM Money</b>
	<form action="checkout_bbmmoney.php" method="POST" id="payment-form">
      <button id="submit-button" type="submit">Pay</button>
    </form>
</body>


Charge Transaksi

Untuk melakukan charge transaction menggunakan BBM Money, beberapa langkah harus dilakukan sebagai berikut:

  • payment_type harus di-set menjadi bbm_money.

Tahap ini membahas cara charge transaksi BBM Money di file checkout_bbmmoney.php. Untuk melakukan charge transaksi, kita dapat menggunakan library Veritrans-PHP. Berikut adalah contoh implementasi charge transaksi pada file checkout_bbmmoney.php: 

<?php
include_once('Veritrans.php');

// TODO: change with your actual server_key that can be found on Merchant Administration Portal (MAP)
Veritrans_Config::$serverKey = "< Your Server Key >";
Veritrans_Config::$isProduction = false;

$transaction_details = array(
  'order_id'    => rand(),
  'gross_amount'  => 145000,
);

$customer_details = array(
    'first_name'    => "Andri",
    'last_name'     => "Litani",
    'email'         => "andri@litani.com",
    'phone'         => "081122334455"
    );

// Data yang akan dikirim untuk request charge transaction dengan BBM Money.
$transaction_data = array(
  'payment_type' => 'bbm_money',
  'transaction_details' => $transaction_details,
  'customer_details' => $customer_details
  );

$result = Veritrans_VtDirect::charge($transaction_data);
?>


Pada contoh di atas, kita menggunakan data BBM Money, transaksi, dan pelanggan sebagai parameter charge transaksi. Anda sebenarnya dapat menambahkan data lain saat charge transaksi, seperti: alamat pelanggan, alamat pengiriman, dan informasi barang. Untuk menambahkan data ini, anda dapat melihat Referensi API - API Methods.


Membuat Link BBM Money

Server Veritrans akan mengembalikan respon untuk setiap charge transaksi dalam bentuk JSON. Library Veritrans-PHP akan melakukan konversi JSON menjadi object PHP secara otomatis. Contoh respon JSON yang dikembalikan oleh veritrans adalah sebagai berikut: 

$result = array(
  "status_code" => "201",
  "status_message" => "Success, BBM Money transaction is successful",
  "transaction_id" => "ff05337c-6c94-4f70-8e81-35acd89b688e",
  "order_id" => "201404141421",
  "payment_type" => "bbm_money",
  "transaction_time" => "2014-04-14 16:03:51",
  "transaction_status" => "pending",
  "permata_va_number => "8778999933221111",
  "gross_amount" => "145000.00"
);


Dua informasi penting dalam respon JSON diatas adalah status_code dan permata_va_number. Jika nilai status_code adalah 201, maka proses charge transaksi berhasil disimpan pada server Veritrans dengan status pending. Anda selanjutnya harus membuat link BBM Money dari nomor permata VA yang didapat dari Veritrans.

Wrapping BBM Money

  • Format API BBM Money

    bbmmoney://api/payment/imp?data=URL encoded JSON data


  • Membuat URL encode JSON data

    1. Membuat JSON

       {
   "reference":"8778999933221111", // permata_va_number yang didapat dari respon Veritrans 
   "callback_url": {
     "check_status":"https://example.com/order/check_status.html",
     "before_payment_error":"https://example.com/order/unfinish.html",
     "user_cancel":"https://example.com/order/cancel.html"
     }
 }

    2. Encode JSON

Status Transaksi

Pada saat charge transaksi berhasil, status transaksi tersebut masih pending. Pelanggan anda harus menyelesaikan transaksi dalam waktu maksimal 24 jam. Begitu proses ini selesai, status transaksi akan berubah menjadi success. Anda dapat mengetahui perubahan status transaksi melalui MAP.

HTTP(S) POST

Server Veritrans akan memberikan respon kepada server anda saat pelanggan anda telah melakukan pembayaran. Contoh respon JSON yang dikirimkan oleh Veritrans adalah sebagai berikut:

VT-Link

*Tutorial berikut akan membantu anda melakukan integrasi ke Veritrans dengan VT-Direct. Apabila butuh bantuan, silahkan hubungi kami di support@veritrans.co.id atau kunjungi halaman support kami..

VT-Link memfasilitasi fitur pembayaran di toko online anda dengan me-redirect pelanggan ke halaman pembayaran Veritrans melalui sebuah link.

Overview


Fields yang Diperlukan

Berikut adalah kolom yang harus diisi untuk pembuatan Link:

  1. Order ID
  2. Item Name
  3. Item Price


Apabila transaksi ditujukan untuk pelanggan tertentu, VT-Link akan dikirim ke pelanggan tersebut melalui email.
Kolom tambahan yang harus diisi adalah:

  1. Customer Email
  2. First Name
  3. Last Name
  4. Phone
Peringatan: Note: Expiration date adalah hari terakhir dimana VT-Link dapat digunakan.


Proses Pembayaran

Veritrans akan membuat link yang mengalihkan pelanggan ke halaman pembayaran veritrans untuk menyelesaikan pembayaran.
Semua metode pembayaran yang tersedia untuk merchant akan dapat diakses di halaman pembayaran tersebut.
Setelah transaksi dilakukan, pelanggan akan dialihkan ke URL yang dapat diatur di MAP.

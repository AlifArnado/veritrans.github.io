Fitur Lain

Selain pembayaran dengan kartu kredit normal, saat ini Veritrans memiliki fitur lain untuk pembayaran kartu kredit melalui VT-Web, yaitu:

  1. Transaksi 3D Secure
  2. Installment
  3. BIN Promo

Berikut adalah tutorial untuk melakukan integrasi menggunakan fitur - fitur sesuai dengan kebutuhan anda.

Mengaktifkan fitur VT-Web 3DSecure

Sebelum merchant mengaktifkan fitur 3D Secure, merchant perlu menanyakan terlebih dahulu kepada sales representative Veritrans untuk memastikan apakah MID 3D Secure sudah siap. Jika sudah siap, dan merchant ingin memproses transaksi dengan 3D Secure, maka parameter ”credit_card_3d_secure” : true perlu ditambahkan, sehingga message JSON-nya menjadi: 

  {
      "payment_type": "vtweb",
      "vtweb": {
          "enabled_payments": ["credit_card"],
          "credit_card_3d_secure" :true,
      },
      "transaction_details": {
          "order_id": "order45",
          "gross_amount": 200000
      },
  }

Pembayaran dengan cicilan (Installment)

Untuk mengaktifkan fitur cicilan (installment), merchant terlebih dahulu harus memiliki kesepakatan dengan acquiring bank. Merchant perlu menanyakan terlebih dahulu kepada sales representative Veritrans untuk memastikan apakah MID untuk cicilan sudah siap. Jika sudah siap, dan merchant ingin memproses transaksi dengan cicilan, maka beberapa parameter perlu ditambahkan pada message JSON yang dikirim, hingga menjadi :

  {
      "payment_type": "vtweb",
      "vtweb": {
          "enabled_payments": ["credit_card"],
        "payment_options" : {
          "installment" : {
            "required" : true,
            "installment_terms" : {
               "bni" : [6, 12],
               "mandiri":[6, 12]
          }
        }}

      },
      "transaction_details": {
          "order_id": "order45",
          "gross_amount": 200000
      },
  }

Keterangan :

  • Required : True

    Transaksi tersebut harus menggunakan cicilan.

  • Required : False

    Transaksi tersebut bisa menggunakan cicilan atau pembayaran full

  • Installment_terms

    Merupakan program/jangka waktu cicilan yang sudah disetujui antara merchant dan bank.

  • Untuk sandbox, anda bisa menggunakan "bni" : [6,12] dan "mandiri" : [6,12] untuk test transaksi cicilan.

BIN Promo

Bin promo adalah fitur yang memungkinkan merchant untuk menawarkan promosi berdasarkan bank penerbit kartu kredit tertentu yang dimiliki customer. Untuk melakukan identifikasi bank penerbit kartu, digunakan informasi BIN (Bank Identification Number) yang merupakan 6 digit pertama dari nomor kartu kredit.

Bila merchant ingin memproses transaksi dengan fitur BIN Promo, maka beberapa parameter perlu ditambahkan pada message JSON yang dikirim, hingga menjadi :

  {
      "payment_type": "vtweb",
      "vtweb": {
          "enabled_payments": ["credit_card"],
          "credit_card_bins": ["411111", "422222"],
      },
      "transaction_details": {
          "order_id": "order45",
          "gross_amount": 200000
      },
  }

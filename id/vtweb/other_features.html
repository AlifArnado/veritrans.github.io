Fitur Tambahan

Tutorial berikut akan membantu anda meng-implementasikan fitur tambahan di VT-Web. Apabila butuh bantuan, silahkan hubungi kami di support@veritrans.co.id atau kunjungi halaman support kami.

Saat ini VT-Web memiliki fitur tambahan untuk metode pembayaran dengan kartu kredit, yaitu:

  1. Memilih Metode Pembayaran
  2. Menon-aktifkan 3D Secure
  3. Cicilan
  4. BIN Promo

Fitur transaksi 3D Secure, cicilan, dan BIN Promo merupakan fitur yang tersedia untuk metode pembayaran melalui kartu kredit. Berikut adalah tutorial untuk melakukan integrasi menggunakan fitur - fitur sesuai dengan kebutuhan anda.

  • Memilih Metode Pembayaran
  • Menon-aktifkan 3D Secure
  • Cicilan
  • Bin Promo

Saat ini, ada 8 metode pembayaran yang dapat digunakan melalui VT-Web:

  1. Kartu Kredit
  2. Mandiri ClickPay
  3. CIMB Clicks
  4. Virtual Account
  5. e-Pay BRI
  6. Telkomsel Cash
  7. XL Tunai
  8. Mandiri Bill Payment

Pada bagian VT-Web – Integrasi, kami telah menjelaskan bagaimana mengaktifkan semua metode pembayaran yang anda miliki secara otomatis. Hal ini dapat diimplementasikan dengan tidak menyertakan parameter enabled_payments. Namun, pada kondisi tertentu anda mungkin perlu menampilkan beberapa metode pembayaran saja pada VT-Web. Jika anda menghadapi kondisi ini, silahkan ikuti petunjuk berikut.

Kartu Kredit

Menggunakan contoh transaksi pada bagian Integrasi, contoh kode PHP yang dapat digunakan adalah:

<?php
include_once('Veritrans.php');

...

// Fill transaction details
$transaction = array(
    "vtweb" => array (
          "enabled_payments" => array("credit_card"),
          "credit_card_3d_secure" => true,
        ),
    'transaction_details' => array(
        'order_id' => rand(),
        'gross_amount' => 10000 // no decimal allowed for creditcard
        )
    );

...

?>

Mandiri ClickPay

Mandiri Clickpay merupakan salah satu fitur pembayaran dari mandiri debit. Metode ini memungkinkan pelanggan melakukan transaksi pembayaran dengan memasukkan 16 digit nomor kartu mandiri debit dan Token PIN Mandiri sebagai validasi transaksi.

Untuk mengaktifkan metode pembayaran Mandiri Clickpay (otomatis aktif pada mode Sandbox), merchant perlu:

  1. Daftarkan metode pembayaran Mandiri Clickpay dengan menghubungi sales representative Veritrans.
  2. Menambahkan parameter mandiri_clickpay pada variabel enabled_payments.

Menggunakan contoh transaksi pada bagian Integrasi, contoh kode PHP yang dapat digunakan adalah:

<?php
include_once('Veritrans.php');

...

// Fill transaction details
$transaction = array(
    "vtweb" => array (
          "enabled_payments" => array("mandiri_clickpay")
        ),
    'transaction_details' => array(
        'order_id' => rand(),
        'gross_amount' => 10000
        )
    );

...

?>


Tab Mandiri Clickpay dapat digunakan oleh pelanggan untuk melakukan transaksi. Form beserta petunjuk pembayaran menggunakan Mandiri Clickpay akan ditampilkan pada halaman ini.

CIMB Clicks

Untuk mengaktifkan metode pembayaran CIMB Clicks (otomatis aktif pada mode Sandbox), merchant perlu:

  2. Menambahkan parameter cimb_clicks pada variabel enabled_payments.

Menggunakan contoh transaksi pada bagian Integrasi, contoh kode PHP yang dapat digunakan adalah:

<?php
include_once('Veritrans.php');

Virtual Account

Dengan menggunakan layanan ini, pelanggan anda akan memiliki pilihan untuk melakukan pembayaran melalui transfer bank ke rekening virtual account Bank Permata.

e-Pay BRI

BRI Epay adalah salah satu metode pembayaran online melalui internet banking yang dikeluarkan oleh Bank BRI. Dengan menggunakan metode pembayaran BRI Epay, anda akan diarahkan ke halaman pembayaran BRI untuk melakukan pembayaran. Sebelum menggunakan layanan BRI Epay ini, pelanggan anda harus melakukan registrasi terlebih dahulu. Untuk info lebih lengkap silahkan mengunjungi website e-Pay BRI.

Mandiri Bill Payment

Mandiri bill payment adalah fitur yang disediakan oleh Bank Mandiri agar nasabah mereka lebih mudah melakukan pembayaran dengan menggunakan akun debit. Fitur ini dibuka untuk semua Mandiri e-channel seperti ATM, Internet Banking, dan mobile Banking. Dengan menggunakan e-channel ini, pelanggan dapat membayar tagihan kapan saja (24 jam).

Menon-aktifkan 3D Secure

Kami sangat menyarankan agar fitur 3D secure digunakan pada transaksi kartu kredit anda. Penggunaan 3D Secure merupakan standar keamanan transaksi kartu kredit. 3D Secure mampu menambah lapisan keamanan transaksi kartu kredit melalui tambahan proses autentikasi pihak bank dan pelanggan anda. Jika pada kasus tertentu anda terpaksa untuk menon-aktifkan fitur 3D Secure, anda cukup menambahkan parameter "credit_card_3d_secure" => false saat request URL VT-Web.

Menggunakan transaksi yang sama dengan bagian Integrasi, contoh kode PHP yang dapat digunakan adalah sebagai berikut:

<?php
include_once('Veritrans.php');

... 

$transaction = array(
    "vtweb" => array (
          "enabled_payments" => array("credit_card"),
          // use false to disable 3ds
          "credit_card_3d_secure" => false
        ),
    'transaction_details' => array(
        'order_id' => 7145,
        'gross_amount' => 10000
        )
    );

...

?>

Fitur Cicilan

Untuk mengaktifkan fitur cicilan, merchant harus memiliki kesepakatan dengan acquiring bank terlebih dahulu. Merchant juga perlu menanyakan kepada sales representative Veritrans apakah MID untuk cicilan sudah siap. Jika sudah siap, merchant cukup menambahkan parameter transaksi installment saat request URL VT-Web.

Menggunakan transaksi yang sama dengan bagian Integrasi, contoh kode PHP yang dapat digunakan adalah sebagai berikut:

<?php
include_once('Veritrans.php');

...

$transaction = array(
    "vtweb" => array (
          "enabled_payments" => array("credit_card"),
          "credit_card_3d_secure" => true,
          // Add installment payment option
          "payment_options" => array(
              "installment" => array(
                  "required" => true,
                  "installment_terms" => array(
                      "bni" => array(6,12),
                      "mandiri" => array(6)
                    )
                )
            )
        ),
    'transaction_details' => array(
        'order_id' => rand(),
        'gross_amount' => 10000
        )
    );

...

?>


Fitur cicilan pada VT-Web memiliki beberapa pilihan parameter. Deskripsi parameter yang dapat digunakan:

Atribut Tipe Data Required Deskripsi
required Boolean Ya Status apakah transaksi wajib menggunakan cicilan atau tidak
installment_terms JSON Object Ya Daftar pilihan bank dan masa cicilan


Deskripsi setiap objek yang ada pada installment_terms:

Atribut Tipe Data Required Keterangan
bank String(50) Ya bank yang menyediakan cicilan. Pada sandbox, parameter yang bisa digunakan adalah bni atau mandiri
term Array Integer Ya Daftar masa cicilan. Pada sandbox, elemen array yang valid adalah 6 dan 12

Note: Minimal satu bank dan satu masa cicilan berlaku harus ada pada objek installment_terms.

BIN Promo

BIN (Bank Identification Number) promo adalah fitur yang memungkinkan anda untuk menawarkan promosi berdasarkan bank penerbit kartu kredit tertentu yang dimiliki pelanggan anda. Bank penerbit kartu kredit dapat ditentukan hanya dengan menggunakan BIN. BIN merupakan 6 digit pertama dari kartu kredit. Sebagai alternatif, anda juga boleh menggunakan 1-8 digit pertama kartu kredit yang akan digunakan pada fitur BIN Promo.

Untuk menggunakan fitur ini, anda terlebih dahulu mengumpulkan daftar BIN yang menerima promo. Daftar BIN tersebut menjadi parameter transaksi credit_card_bins, transaksi hanya dapat dilakukan oleh kartu kredit yang termasuk dalam daftar BIN. Gunakan daftar BIN yang anda miliki sebagai nilai dari atribut credit_card_bins saat charge transaksi.

Menggunakan transaksi yang sama dengan bagian Integrasi, contoh kode PHP yang dapat digunakan adalah sebagai berikut:

<?php
include_once('Veritrans.php');

...

$transaction = array(
    "vtweb" => array (
          "enabled_payments" => array("credit_card"),
          "credit_card_3d_secure" => true,
          //Register BIN list
          "credit_card_bins" => array("48111111", "3111", "5")
        ),
    'transaction_details' => array(
        'order_id' => 8033,
        'gross_amount' => 10000
        )
    );

$vtweb_url = Veritrans_Vtweb::getRedirectionUrl($transaction);

...

?>

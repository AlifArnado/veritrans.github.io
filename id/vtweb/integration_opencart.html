Tahapan Integrasi VT-Web OpenCart

  1. Instalasi Plugin
  2. Tes Transaksi


  • Tahap 1Instalasi Plugin
  • Tahap 2Tes Transaksi

Apa yang Diperlukan

  1. Sebuah toko online berbasis OpenCart. (Plugin ini di-test dengan OpenCart versi 1.5.6.4)
  2. PHP versi minimal 5.3.
  3. Plugin Veritrans untuk OpenCart [ Github | Zip ].

Proses Instalasi

  1. Ekstrak file vtweb-opencart-master.zip yang sudah di-download sebelumnya.
  2. Di dalam folder zip tersebut terdapat tiga folder (admin, catalog, dan system). Upload folder tersebut ke dalam direktori OpenCart anda.

Konfigurasi Plugin

Untuk melakukan konfigurasi plugin Veritrans-OpenCart:

  1. Masuk ke halaman admin OpenCart anda.
  2. Buka menu Extension - Payments pada halaman administrasi OpenCart.
  3. Cari lalu install module Veritrans untuk mengaktifkan plugin.
  4. Pada modul Veritrans klik Edit untuk masuk ke halaman plugin.
  5. Masukkan data-data yang diperlukan seperti gambar di bawah ini.



  6. Klik Save untuk menyimpan konfigurasi.

  7. Login ke Settings - Configuration pada MAP untuk mengubah konfigurasi berikut:

    URL Role Redirect URL
    Payment Notification URL http://[your shop’s homepage]/index.php?route=payment/veritrans/payment_notification
    Finish Redirect URL http://[your shop’s homepage]/index.php?route=checkout/success&
    Error Redirect URL http://[your shop’s homepage]/index.php
    Unfinish Redirect URL http://[your shop’s homepage]/index.php


  1. Lakukan transaksi sukses di toko Anda dengan menggunakan nomor kartu kredit berikut (sandbox mode):

    Jenis Visa
    Nomor Kartu 4811 1111 1111 1114
    CVV 123
    Exp. Month 01
    Exp. Year 2020


  2. Cek beberapa poin berikut untuk memastikan plugin sudah ter-install dan bekerja dengan baik.

    Cek Point Hasil yang diharapkan Bila hasil tidak sesuai dengan yang diharapkan
    Cek order status di back end OpenCart Order status berubah menjadi processing Cek setting payment notification URL di MAP
    Merchant Dapat Jika tidak, cek setting di MAP
    Customer mendapat email notifikasi Dapat Jika tidak, cek setting di MAP

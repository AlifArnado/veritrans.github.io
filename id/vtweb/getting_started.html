Langkah Awal

Tujuan: Merchant mengetahui apa yang perlu dipersiapkan sebelum melakukan integrasi ke VT-Web

Ada beberapa hal yang harus dipersiapkan sebelum kita menggunakan VT-Web:

  1. Membuat akun Sandbox di Veritrans.

    Digunakan untuk development. Data dan transaksi yang digunakan pada akun ini bersifat dummy. Pada tahap pengembangan, diharuskan untuk menggunakan akun sandbox. Akun sandbox dapat dibuat di sini.

  2. Mengisi informasi dasar tentang anda di MAP. Dapat ditemukan di menu Settings - General Settings.

  3. Mengunduh library Veritrans-PHP [ Github | Zip ]

  4. Mencatat server key dan client key akun anda. Kedua key tersebut dapat diperoleh dengan

    • Login Sandbox MAP (Merchant Administration Portal).

    • Pilih menu Settings dan Access keys

    • Perhatian!!! Jaga baik-baik kerahasiaan server key anda.

  5. Menangani redirect URL

    Tahap terakhir sebelum memulai integrasi adalah menentukan bagaimana pelanggan anda kembali ke website setelah proses pembayaran selesai dilakukan. Setelah pelanggan anda menyelesaikan proses pembayaran, mereka akan dialihkan kembali ke website anda. Kemana pelanggan anda akan dialihkan, dapat diatur melalui MAP pada menu Settings - Configuration.

    Berikut adalah alur kemana pelanggan anda akan dialihkan sesaat setelah proses pembayaran selesai dilakukan.

    1. Finish Redirect URL
      2. Alamat URL kemana pelanggan anda akan diarahkan jika proses pembayaran berhasil
    2. Error Redirect URL
      3. Alamat URL kemana pelanggan anda akan diarahkan jika pembayaran gagal
    3. Unfinish Redirect URL
      4. Alamat URL kemana pelanggan anda akan diarahkan saat menekan tombol "Back to Merchant Website" (Batal melakukan proses pembayaran).

Jika semua poin di atas selesai dilakukan, integrasi VT-Web dapat dimulai.

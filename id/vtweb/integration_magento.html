Tahapan Integrasi VT-Web Magento

  1. Instalasi Plugin
  2. Tes Transaksi


  • Instalasi Plugin
  • Tes Transaksi

Apa yang Diperlukan

  1. Sebuah toko online berbasis Magento. (Plugin ini di-test dengan Magento versi 1.9)
  2. PHP versi minimal 5.3.
  3. Plugin Veritrans untuk Magento [ Github | Zip ].

Proses Instalasi

  1. Ekstrak file VTWebmagento-master.zip yang sudah di-download sebelumnya.
  2. Upload / copy folder app, lib, dan skin yang baru diekstrak ke dalam direktori Magento anda.

Konfigurasi Plugin

Untuk melakukan konfigurasi plugin Veritrans-Magento:

  1. Masuk ke halaman admin Magento anda.
  2. Buka menu System - Configuration pada halaman administrasi Magento.
  3. Pada bagian Sales-Payment Methods cari tab Veritrans. Bila tab Veritrans belum ada, klik Save Config terlebih dahulu.
  4. Klik general untuk memulai konfigurasi.

  5. Masukkan data-data yang diperlukan seperti di bawah ini kemudian klik Save Config.

  6. Login ke portal MAP untuk mengubah isi pada beberapa opsi berikut:

    URL Role Redirect URL
    Payment Notification URL http://[your_shop’s_homepage]/vtweb/payment/notification
    Finish Redirect URL http://[your_shop’s_homepage]/
    Error Redirect URL http://[your_shop’s_homepage]/
    Unfinish Redirect URL http://[your_shop’s_homepage]/

Di tahap ini, kita akan mencoba melakukan tes transaksi. Jika anda menggunakan akun sandbox, berbagai macam skenario transaksi dapat diuji coba, daftar skenario beserta informasi akun yang tersedia untuk uji coba dapat dilihat di sini.

Anda sangat disarankan untuk melakukan tes transaksi menggunakan VT-Web. Manfaat utama melakukan tes transaksi adalah:

  1. Memahami flow pembayaran VT-Web
  2. Mengetahui status pembayaran melalui MAP (Merchant Administration Portal) dan notifikasi

Saat ini, kita kita akan mencoba melakukan transaksi kartu kredit 3D Secure yang berhasil. Berdasarkan Referensi - Daftar Kartu Uji Coba, kartu kredit yang dapat digunakan adalah:

Jenis Visa
Nomor Kartu 4811 1111 1111 1114
CVV 123
Exp. Month 01
Exp. Year 2020

Bila Setting plugin sudah benar, maka akan tampil halaman pembayaran VT-Web.

Cek beberapa poin berikut setelah transaksi diatas berhasil dilakukan untuk memastikan plugin sudah ter-install dan bekerja dengan baik.

Cek Point Hasil yang diharapkan Bila hasil tidak sesuai dengan yang diharapkan
Cek order status di back end OpenCart Order status berubah menjadi processing Cek setting endpoint/payment notification URL di MAP
Merchant mendapatkan email notifikasi Dapat Cek setting di MAP, menu setting - email notification
Customer mendapat email notifikasi Dapat Cek setting di MAP, menu setting - email notification

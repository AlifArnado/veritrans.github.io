BIN Promo

BIN (Bank Identification Number) merupakan 6 digit pertama dari nomor kartu kredit. Bank penerbit kartu kredit dapat ditentukan dengan menggunakan BIN, BIN promo memungkinkan charge transaksi hanya dapat dilakukan oleh kartu kredit tertentu. Setiap BIN yang anda gunakan dapat direpresentasikan oleh 1 hingga 8 digit pertama nomor kartu kredit.

Spesifikasi fungsi BIN Promo sama dengan spesifikasi charge transaksi. Karena petunjuk implementasi BIN Promo pada VT-Web dan VT-Direct berbeda, pembahasan akan dilakukan secara terpisah.

  • VT-Web
  • VT-Direct

Sama dengan charge transaksi VT-Web, output dari fungsi BIN promo adalah URL pembayaran. Pelanggan harus anda redirect ke URL pembayaran VT-Web untuk menyelesaikan transaksi. Implementasi BIN Promo pada VT-Web sangat sederhana. Anda cukup menambahkan atribut credit_card_bins pada JSON request charge transaksi. Berikut adalah JSON request BIN Promo pada VT-Web:

{
    // Required
    "payment_type" : "vtweb",
    // Required
    "vtweb" : {
        "credit_card_3d_secure" : true,
        "enabled_payments": ["credit_card"],
        "credit_card_bins": ["48111111", "3111", "5"]
    },
    // Required
    "transaction_details" : {
	  "order_id" : "M17550",
	  "gross_amount" : 145000
	},
    // Optional
    "customer_details" : {
	    "first_name" : "Andri",
	    "last_name" : "Litani",
	    "email" : "andri@litani.com",
	    "phone" : "081122334455",
	    // Optional
	    "billing_address" : {
			    "first_name" : "Andri",
			    "last_name" : "Litani",
			    "address" : "Mangga 20",
			    "city" : "Jakarta",
			    "postal_code" : "16602",
			    "phone" : "081122334455",
			    "country_code" : "IDN"
		    },
		// Optional
	    "shipping_address" : {
			    "first_name" : "Obet",
			    "last_name" : "Supriadi",
			    "address" : "Manggis 90",
			    "city" : "Jakarta",
			    "postal_code" : "16601",
			    "phone" : "08113366345",
			    "country_code" : "IDN"
		    }
	    },
    // Optional
    "item_details" : [
		{
		    "id" : "a1",
		    "price" : 50000,
		    "quantity" : 2,
		    "name" : "Apel"
		},
		{
		    "id" : "a2",
		    "price" : 45000,
		    "quantity" : 1,
		    "name" : "Jeruk"
		}

    ]
}

Struktur JSON response fungsi BIN Promo sama dengan charge transaksi VT-Web. Contoh JSON response sukses:

{
    "status_code": "201",
    "status_message": "OK, success do VTWeb transaction, please go to redirect_url",
    "redirect_url": "https://vtweb.sandbox.veritrans.co.id/v2/vtweb/550ad19d-e01b-4c8f-945a-67902c7b83da"
}

Pada VT-Direct, anda harus memperoleh token ID terlebih dahulu. Token ID merepresentasikan informasi kartu kredit pelanggan anda yang telah disimpan di server Veritrans. Cara memperoleh Token ID pada BIN Promo sama dengan charge transaksi normal.

Setelah token diperoleh, charge transaksi BIN Promo dapat dilakukan dengan menambahkan atribut bins dalam credit_card:

{
    // Required
    "payment_type" : "credit_card",
    // Required
    "credit_card" : {
        "bank" : "bni",
        "token_id": "< your token ID >",
        "bins" : ["48111111", "3111", "5"]
    },
    // Required
    "transaction_details" : {
	  "order_id" : "P17550",
	  "gross_amount" : 145000
	},
    // Required
    "customer_details" : {
	    "first_name" : "Andri",
	    "last_name" : "Litani", // Optional
	    "email" : "andri@litani.com",
	    "phone" : "081122334455",
        // Optional
	    "billing_address" : {
			    "first_name" : "Andri",
			    "last_name" : "Litani",
			    "address" : "Mangga 20",
			    "city" : "Jakarta",
			    "postal_code" : "16602",
			    "phone" : "081122334455",
			    "country_code" : "IDN"
		    },
        // Optional
	    "shipping_address" : {
			    "first_name" : "Obet",
			    "last_name" : "Supriadi",
			    "address" : "Manggis 90",
			    "city" : "Jakarta",
			    "postal_code" : "16601",
			    "phone" : "08113366345",
			    "country_code" : "IDN"
		    }
	    },
    // Optional
    "item_details" : [
		{
		    "id" : "a1",
		    "price" : 50000,
		    "quantity" : 2,
		    "name" : "Apel"
		},
		{
		    "id" : "a2",
		    "price" : 45000,
		    "quantity" : 1,
		    "name" : "Jeruk"
		}

    ]
}

Pada contoh di atas, BIN yang didaftarkan adalah 48111111, 3111, dan 5. Jika kartu kredit dengan BIN "411111" coba digunakan, charge transaksi akan gagal.

Struktur JSON response BIN Promo sama dengan charge transaksi VT-Direct kartu kredit. Perbedaan hanya terdapat konten status_message. Contoh JSON response transaksi sukses: 

{
    "transaction_id": "1a1a66f7-27a7-4844-ba1f-d86dcc16ab27",
    "order_id": "P17550",
    "gross_amount": "145000.00",
    "payment_type": "credit_card",
    "transaction_time": "2014-08-24 15:39:22",
    "transaction_status": "capture",
    "fraud_status": "accept",
    "masked_card": "481111-1114",
    "status_code": "200",
    "status_message": "OK, Credit Card 3D Secure transaction is successful",
    "approval_code": "1408869563148"
}

Contoh JSON response transaksi gagal:

{
    "status_code": "400",
    "status_message": "Validation Error, [credit card number does not match with the list of credit_card.bins]",
    "validation_messages": [
        "credit card number does not match with the list of credit_card.bins"
    ]
}

Mendapatkan Status Transaksi

Keterangan Mendapatkan informasi status dari suatu transaksi dengan order_id tertentu
Endpoint BASE_URL/{order_id}/status
HTTP Method GET
HTTP Header Accept: application/json
Authorization: Basic base64(server_key:)
Body Request Blank


Contoh respon sukses

{
    "status_code": "200",
    "status_message": "Success, transaction found",
    "transaction_id": "e3b8c383-55b4-4223-bd77-15c48c0245ca",
    "masked_card": "481111-1114",
    "order_id": "2014112112",
    "payment_type": "credit_card",
    "transaction_time": "2014-11-21 13:07:50",
    "transaction_status": "settlement",
    "fraud_status": "accept",
    "approval_code": "1416550071152",
    "signature_key": "4ef8218aad5b64bae2ec9d6b0f0a0b059b88bd298f9e79e662f641bae7cd24992fb67547ea60cc3f9a820ca7e5649cf5e1f0a8e4b5ef24001f951b7a9d1a8f42",
    "bank": "mandiri",
    "gross_amount": "10000.00"
}


Contoh respon gagal

Approve transaksi

Keterangan Meng-approve transaksi dengan order_id tertentu yang telah mendapatkan status challenge dari Fraud Detection System.
Endpoint BASE_URL/{order_id}/approve
HTTP Method POST
HTTP Header Accept: application/json
Authorization: Basic base64(server_key:)
Body Request Blank


Contoh respon sukses

{
    "status_code": "200",
    "status_message": "Success, transaction is approved",
    "transaction_id": "2af158d4-b82e-46ac-808b-be19aaa96ce3",
    "masked_card": "451111-1117",
    "order_id": "2014112112",
    "payment_type": "credit_card",
    "transaction_time": "2014-11-27 10:05:10",
    "transaction_status": "capture",
    "fraud_status": "accept",
    "approval_code": "1417057511311",
    "bank": "bni",
    "gross_amount": "10000.00"
}


Contoh respon gagal

{
    "status_code": "404",
    "status_message": "The requested resource is not found"
}


Cancel transaksi

Keterangan Membatalkan transaksi dengan order_id tertentu. Pembatalan hanya bisa dilakukan jika transaksi tersebut belum melalui proses settlement.
Endpoint BASE_URL/{order_id}/cancel
HTTP Method POST
HTTP Header Accept: application/json
Authorization: Basic base64(server_key:)
Body Request blank


Contoh respon sukses

{
    "status_code": "200",
    "status_message": "Success, transaction is canceled",
    "transaction_id": "2af158d4-b82e-46ac-808b-be19aaa96ce3",
    "masked_card": "451111-1117",
    "order_id": "2014112112",
    "payment_type": "credit_card",
    "transaction_time": "2014-11-27 10:05:10",
    "transaction_status": "cancel",
    "fraud_status": "accept",
    "bank": "bni",
    "gross_amount": "10000.00"
}


Contoh respon gagal

{
    "status_code": "404",
    "status_message": "The requested resource is not found"
}

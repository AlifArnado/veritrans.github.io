Get transaction status

Description Mendapatkan informasi status dari suatu transaksi dengan id (order_id) tertentu.
Endpoint Sandbox : https://api.sandbox.veritrans.co.id/v2/{id}/status
Production : https://api.veritrans.co.id/v2/{id}/status
HTTP Method GET
HTTP Header Accept: application/json
Content-Type: application/json
Authorization: Basic base64(server_key:)
Body Request blank

Approve transaction

Description Meng-approve transaksi dengan id (order_id) tertentu yang telah mendapatkan status ‘challenge’ dari Fraud Detection System.
Endpoint Sandbox : https://api.sandbox.veritrans.co.id/v2/{id}/approve
Production : https://api.veritrans.co.id/v2/{id}/approve
HTTP Method POST
HTTP Header Accept: application/json
Content-Type: application/json
Authorization: Basic base64(server_key:)
Body Request blank

Cancel transaction

Description Membatalkan transaksi dengan id (order_id) tertentu. Pembatalan hanya bisa dilakukan jika transaksi tersebut belum melalui proses settlement.
Endpoint Sandbox : https://api.sandbox.veritrans.co.id/v2/{id}/cancel
Production : https://api.veritrans.co.id/v2/{id}/cancel
HTTP Method POST
HTTP Header Accept: application/json
Content-Type: application/json
Authorization: Basic base64(server_key:)
Body Request blank

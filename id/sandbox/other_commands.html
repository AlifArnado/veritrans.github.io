Mendapatkan Status Transaksi

Description Mendapatkan informasi status dari suatu transaksi dengan order_id tertentu
Endpoint BASE_URL/{order_id}/status
HTTP Method GET
HTTP Header Accept: application/json
Authorization: Basic base64(server_key:)
Body Request Blank

Contoh JSON response

{
    "status_code": "200",
    "status_message": "OK, transaction found",
    "transaction_id": "b833d18c-2670-4659-9a01-1d41d0fb3a3d",
    "order_id": "K17550",
    "payment_type": "credit_card",
    "transaction_time": "2014-08-25 09:56:58",
    "transaction_status": "capture",
    "fraud_status": "accept",
    "masked_card": "481111-1114",
    "gross_amount": "145000.00"
}

Approve transaksi

Description Meng-approve transaksi dengan order_id tertentu yang telah mendapatkan status challenge dari Fraud Detection System.
Endpoint BASE_URL/{order_id}/approve
HTTP Method POST
HTTP Header Accept: application/json
Authorization: Basic base64(server_key:)
Body Request Blank

Contoh respon sukses

{
    "status_code": "201",
    "status_message": "OK, transaction has been approved",
    "transaction_id": "7f08cc57-14e3-40e8-830e-2fbc93773d6f",
    "order_id": "17550",
    "payment_type": "credit_card",
    "transaction_time": "2014-08-25 11:52:58",
    "transaction_status": "capture",
    "fraud_status": "accept",
    "masked_card": "451111-1117",
    "gross_amount": "145000.00"
}

Cancel transaksi

Description Membatalkan transaksi dengan order_id tertentu. Pembatalan hanya bisa dilakukan jika transaksi tersebut belum melalui proses settlement.
Endpoint BASE_URL/{order_id}/cancel
HTTP Method POST
HTTP Header Accept: application/json
Authorization: Basic base64(server_key:)
Body Request blank

Contoh respon sukses

{
    "status_code": "200",
    "status_message": "OK, transaction has been cancelled",
    "transaction_id": "bffa405b-fe66-417e-92b4-8fccf82967fa",
    "order_id": "37550",
    "payment_type": "credit_card",
    "transaction_time": "2014-08-25 11:57:53",
    "transaction_status": "cancel",
    "fraud_status": "challenge",
    "masked_card": "481111-1114",
    "gross_amount": "145000.00"
}

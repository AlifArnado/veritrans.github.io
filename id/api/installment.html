Cicilan

Cicilan merupakan salah satu varian dari charge transaksi yang memungkinkan pembayaran dilakukan secara berkala dalam jangka waktu tertentu. Cicilan hanya berlaku untuk transaksi melalui kartu kredit. Spesifikasi fungsi cicilan sama dengan spesifikasi charge transaksi. Karena petunjuk implementasi cicilan pada VT-Web dan VT-Direct berbeda, pembahasan akan dilakukan secara terpisah.

  • VT-Web
  • VT-Direct


Sama dengan charge transaksi VT-Web, output dari fungsi cicilan adalah URL pembayaran. Pelanggan harus anda redirect ke URL pembayaran VT-Web untuk menyelesaikan transaksi. Implementasi fungsi cicilan pada VT-Web sangat sederhana, anda cukup menambahkan atribut payment_options dan installment pada JSON request charge transaksi. Berikut adalah JSON request cicilan pada VT-Web: 

{
	// Required
	"payment_type": "vtweb",
	// Required
	"vtweb": {
		"credit_card_3d_secure": true,
		"enabled_payments": ["credit_card"],
		"payment_options": {
			"installment": {
				"required": true,
				"installment_terms": {
					"bni": [],
					"mandiri":[6, 12]
				}
			}
		}
	},
	// Required
	"transaction_details": {
		"order_id": "K17550",
		"gross_amount": 145000
	},
	// Optional
	"customer_details": {
		"first_name": "Andri",
		"last_name": "Litani",
		"email": "andri@litani.com",
		"phone": "081122334455",
	// Optional
	"billing_address": {
		"first_name": "Andri",
		"last_name": "Litani",
		"address": "Mangga 20",
		"city": "Jakarta",
		"postal_code": "16602",
		"phone": "081122334455",
		"country_code": "IDN"
		},
	// Optional
	"shipping_address": {
		"first_name": "Obet",
		"last_name": "Supriadi",
		"address": "Manggis 90",
		"city": "Jakarta",
		"postal_code": "16601",
		"phone": "08113366345",
		"country_code": "IDN"
		}
	},
	// Optional
	"item_details": [
		{
			"id": "a1",
			"price": 50000,
			"quantity": 2,
			"name": "Apel"
		},
		{

			"price": 45000,
			"quantity": 1,
			"name": "Jeruk"
		}
	]
}


Perlu diperhatikan jika masa cicilan pada bank BNI adalah array kosong, sesuai keterangan pada bagian spesifikasi, semua masa cicilan yang valid akan ditampilkan pada VT-Web.

Struktur JSON response fungsi cicilan sama dengan charge transaksi VT-Web. Contoh JSON response sukses: 

{
	"status_code": "201",
	"status_message": "OK, success do VTWeb transaction, please go to redirect_url",
	"redirect_url": "https://vtweb.sandbox.veritrans.co.id/v2/vtweb/550ad19d-e01b-4c8f-945a-67902c7b83da"
}


Pada VT-Direct, anda harus memperoleh token ID terlebih dahulu. Token ID merepresentasikan informasi kartu kredit pelanggan anda yang telah disimpan di server Veritrans. Untuk memperoleh token, anda dapat mengikuti petunjuk di VT-Direct – Mendapatkan Token. Perlu diperhatikan bahwa parameter request token untuk installment berbeda dengan token transaksi normal. Anda harus menambahkan installment dan installment_term pada parameter.

Setelah token diperoleh, charge transaksi cicilan dapat dilakukan dengan menambahkan atribut installment_term dalam credit_card: 

{
	// Required
	"payment_type": "credit_card",
	// Required
	"credit_card": {
		"bank": "bni",
		"token_id": "< your installment token ID >",
		"installment_term": 12
	},
	// Required
	"transaction_details": {
		"order_id": "K17550",
		"gross_amount": 145000
	},
	// Required

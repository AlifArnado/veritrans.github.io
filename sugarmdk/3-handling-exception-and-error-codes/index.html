    • 3. Handling Exception and Error Codes

    Every service that is provided by Veritrans Managers, will throw a VeritransException object if an error occurred. From the VeritransException object, the message, status and the error codes can be obtained.

    VeritransException Object

    Property Description
    code Error code of the exception occurred
    status Short description of the exception occurred
    message Complete description of the exception occurred

    Sample Code: Handle exception for normal charge

    try {
    ChargeResponse response = paymentManager.doCharge(chargeReq, sMerchant);
} catch (VeritransException e) {
    e.getStatus();
    e.getCode();
    e.getMessage();
}
}

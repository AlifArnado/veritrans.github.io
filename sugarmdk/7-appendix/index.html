    • A1. Response Objects Parameters Definition

    ChargeResponseObject

    Used by Normal Charge, Installment Charge, Mandiri ClickPay Charge

    Class Property Data Type Description
    ChargeResponse data Object:ChargeData Data object, consist all the response data attributes
    code String Response code. Refer to Response Code List
    message String Message response
    status String “success”, “failure”, or “challenge” (fyi, challenge only applies to credit card)
    ChargeData orderId AN Must be Unique
    grossAmount Long Total amount (must be the same value with individual item * price)
    email ANS Email address
    items List : Item Processed item list
    shippingAddress Object : Address Shipping address
    billingAddress Object : Address Billing address
    cardNumber N Card number
    transactionStatus Enum :TransactionStatus The status of transaction. Refer to Transaction Status Table
    transactionTime DateTime Date time details of transaction
    transactionType Enum: TransactionType Type of the transaction
    bank Enum: Bank Bank related to the transaction
    Item Id AN Id of Item
    price Long Price of item > 0
    quantity int Quantity of item >= 1
    name ANS Item name
    subtotal Long Subtotal value (quantity * price)
    tenor String Tenor for installment item. Only used in Internet Banking (BCA ClickPay Installment).
    codePlan String Code plan for installment item. Currently only support for “000”. Only used in Internet Banking (BCA ClickPay Installment).
    mId String Credit card MID based on product. Only used in Internet Banking (BCA ClickPay Installment).
    Address firstName ANS First name of address' owner
    lastName ANS Last name of address' owner
    address1 ANS Acceptable symbols - _ ‘ , . @ ( ) /
    address2 ANS Acceptable symbols - _ ’ , . @ ( ) /
    city AS City of the address. Accepted Symbols : _ ‘ , . @ -
    postalCode N >= 1
    phone NS Acceptable symbols : +, -
    countryCode A For now only support :“IDN”

    Legend: AN : Alphanumeric, ANS : Alphanumeric Symbol, NS : Numeric Symbol, N : Numeric

    D3SecureResponse Object

    Used by 3D Secure Charge

    Class Property Data Type Description
    D3SecureResponse merchantId AN Merchant id
    merchantMessage ANS Merchant personalized message (will be set by veritrans for now)
    orderId ANS In processed orderId
    d3Eci A Electronic Commerce Identifier
    d3TransactionId AN 3D Secure transactionId
    resultCode A Result code from bank
    transactionStatus Enum :TransactionStatus The status of transaction. Refer to Transaction Status Table
    transactionTime DateTime Date time of transaction

    Legend: AN : Alphanumeric, ANS : Alphanumeric Symbol, NS : Numeric Symbol, N : Numeric

    CancelResponse Object

    Used by Cancel Credit Card Transaction

    Class Property Data Type Description
    CancelResponse data Object: CancelData Data containing the details of credit card transaction cancellation
    status String Status of the operation
    message String Message regarding the status
    code String Response Code. Refer to Response Code List
    CancelData orderId String Identification of the order cancelled
    transactionStatus String Status of the transaction. Should be TransactionStatus.CANCEL. Refer to Transaction Status Table
    transactionTime Object: Date Time of the cancellation

    Legend: AN : Alphanumeric, ANS : Alphanumeric Symbol, NS : Numeric Symbol, N : Numeric

    ChallengeActionResponse Object

    Used by Accept Challenge Transaction, Deny Challenge Transaction

    Class Property Data Type Description
    ChallengeActionResponse parent Response Parent Response
    status Enum :ChallengeActionStatus Status of the challenge action. The result could be : ChallengeActionStatus.SUCCESS , ChallengeActionStatus.FAILED
    code String Status code of the challenge action
    message String Message status of the challenge action

    Legend: AN : Alphanumeric, ANS : Alphanumeric Symbol, NS : Numeric Symbol, N : Numeric

    OrderResponse Object

    Used by Get Order by Id

    Class Property Data Type Description
    OrderResponse customerInformation Object:CustomerInformation Object of customer information class
    shippingInformation Object:ShippingInformation Object of shipping information class
    transactionDetail Object:TransactionDetail Object of transaction detail class
    items Object:List List of object order items
    grossAmount Long Gross amount of the particular order
    history Object: List List of every transaction history on the order id
    CustomerInformation cardNumber String The card number used in the order
    expireDate String Date expiration detail of the card used in the order
    customerId String Identification number for customer
    ShippingInformation name String Name of the customer making the order
    phone String Phone number of the customer making the order
    email String Email address of the customer making the order
    address String Address of the customer making the order
    TransactionDetail orderId String Order id
    tradeTime Date Timestamp of the start transaction
    amount Long Amount of the transaction
    bank Enum:Bank The acquiring bank used for the transaction
    transactionType Enum:TransactionType The type of payment used. Refer to Transaction Types Table
    transactionStatus Enum :TransactionStatus Status of transaction. Refer to Transaction Status Table
    OrderItem id String Commodity id of the item
    name String Product name
    quantity Integer Quantity of the item bought in the order
    price Double The price per unit of the item
    subtotal Double Subtotal of the Order Item (price * quantity)
    OrderHistory tradeTime String Time when the transaction occurred
    status String Order status
    command String Used by gateway for defining which command to be called.
    mStatus String Individual transaction status of merchant
    resultCode String Response Code. Refer Response Code List

    Legend: AN : Alphanumeric, ANS : Alphanumeric Symbol, NS : Numeric Symbol, N : Numeric

    OrderSearchResponse Object

    Used by Get Order List by Multiple Order Ids, Get Order by Date Range, Transaction Status, and Merchant ID

    Class Property Data Type Description
    OrderSearchResponse payments Object:List List of object Payment as search response
    payment orderId String Order id
    merchantId String Merchant id related to the order
    grossAmount Long Gross amount of the particular order
    customerName String Name of the customer
    customerEmail String Email of the customer
    transactionStatus Enum :TransactionStatus Status of transaction. Refer to Transaction Status Table
    customerPhone String Phone number of the customer
    transactionTime Date Time when the transaction occurred
    transactionType Enum:TransactionType The type of payment used in this internet banking order. Refer to Transaction Types Table
    bank Enum :Bank Enumeration of Bank Type

    Legend: AN : Alphanumeric, ANS : Alphanumeric Symbol, NS : Numeric Symbol, N : Numeric

    SubMerchant Object

    Used by Normal Charge, 3D Secure Charge, Installment Charge, Cancel Transaction, Create SubMerchant, Update SubMerchant, Get SubMerchant

    Class Property Data Type Description
    SubMerchant parent Merchant Parent merchant
    Id AN Merchant identifier
    username AN Username of merchant
    password AN Password of merchant
    name AN Merchant name
    email ANS Billing email address
    ownerName AN Name of the merchant owner/contact
    phone NS Phone number of the owner/contact
    sectorType int The sector of the industry. Refer to Sector Types Table.
    npwp N Tax identifier number
    clientKey ANS Client key for gateway authentication
    serverKey ANS Server key for gateway authentication

    Legend: AN : Alphanumeric, ANS : Alphanumeric Symbol, NS : Numeric Symbol, N : Numeric

    InternetBankingResponse Object

    Used by Mandiri ClickPay, CIMB Clicks, BCA KlikPay, KlikBCA

    Class Property Data Type Description
    InternetBankingResponse merchantId String Merchant identifier
    orderId String Order id
    webPageSelector String Code of the bank = 01: CIMB Clicks , 03: Mandiri ClickPay , 04: BCA KlikPay
    amount String Total amount for the order
    currency String Currency used
    productDesc String Description for the order. Only used in BCA ClickPay
    email String Email of user
    payType String Type of the payment. Only used in BCA ClickPay. 01: Full Transaction , 02: Installment Transaction , 03: Combination of 01 and 02
    miscFee String Miscellaneous fee. Including VAT, transportation tax, etc. Only used in BCA ClickPay
    cardNumber String Card number. Only used in Mandiri ClickPay
    transactionType Enum:TransactionType The type of payment used in this internet banking order. Refer to Transaction Types Table
    transactionStatus Enum:TransactionStatus Status of the transaction. Refer to Transaction Status Table
    transactionDate Object: Date Date when the transaction takes place.
    address Object:Address Address related properties of the transaction. Refer to Address Object
    items Collection Item list in the transaction. Refer to Item Object
    responseCode String Code of the response
    responseMessage String Message of the response

    Legend: AN : Alphanumeric, ANS : Alphanumeric Symbol, NS : Numeric Symbol, N : Numeric

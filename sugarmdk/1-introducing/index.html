    • The Sugar Merchant Development Kit (MDK), is a development kit developed for use by Rakuten Belanja Online in Indonesia. Sugar MDK includes functions specific to the needs of online shopping malls, such as functions to allow the Master merchant to add and register sub-merchants.

    1.1 Getting started with Sugar MDK

    Please include the “sugar-mdk.jar” to utilize the Sugar MDK. The latest library is currently: sugar-mdk-0.1.8.jar

    1.2 Instantiating VeritransGateway Object

    The first thing to do to gain access to Veritrans' services is by instantiating the VeritransGateway Object.

    Veritrans gateway = VeritransGatewayFactory.createInstance(username, password);
PaymentManager paymentManager = gateway.getPaymentManager();

    Username and Password is needed to create the VeritransGateway object using the factory object. Username and Password for testing and production purposes will be provided by Veritrans.

    1.3 Veritrans Manager Objects

    Veritrans has separated the Services provide by the Sugar MDK into different managers. Each manager has a specific function for a domain that it can perform.

    Status Description
    Payment Manager All payment related services. Refer to Section 2
    Order Manager All order related services. Refer to Section 3
    SubMerchantManager All sub-merchant management related services. Refer to Section 4 
    PaymentManager     paymentManager     = gateway.getPaymentManager();
OrderManager       orderManager       = gateway.getOrderManager();
SubMerchantManager subMerchantManager = gateway.getSubMerchantManager();

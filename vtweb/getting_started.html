Getting Started

Target: Merchant understands what to prepare before integration with VT-Web

There are a few stuffs need to be prepared before integrating with VT-Web:

  1. Create Veritrans Sandbox account.

    Utilized for development. Data and transaction made on this account is dummy in nature. During development period, merchant has to use sandbox account. Sandbox account can be created here.

  2. Fill in your basic information in MAP (Merchant Administration Portal). Can be found in menu Settings - General Settings.

  3. Download Veritrans-PHP library [ Github | Zip ]

  4. Take note the server key and client key of your account. Both keys can be obtained by

    • Login Sandbox MAP (Merchant Administration Portal).

    • Choose menu Settings and Access keys

    • Caution!!! Be discreet on your server key.

  5. Manage redirect URL

    The last step is to decide on how your customer will return to your website after payment process is completed. The website page where customer will be redirected can be configured in MAP or code.

    To set through MAP, you can go to Settings - Configuration menu on MAP.

    To set through code, use the following parameters:

     <?php 
 ...

 // Fill transaction data
 $transaction = array(
 "vtweb" => array (
       "credit_card_3d_secure" => true,
       'finish_redirect_url' => 'http://www.example.com',
       'unfinish_redirect_url' => 'http://www.example.com',
       'error_redirect_url' => 'http://www.example.com'
     ),
 'transaction_details' => array(
     'order_id' => rand(),
     'gross_amount' => 10000,
     )
 );

 ...
 ?>

    Detailed below is the flow on how the customer will be redirected after payment process is completed.

    1. Finish Redirect URL
      2. URL address where customer will be redirected if payment process is successful
    2. Error Redirect URL
      3. URL address where customer will be redirected if payment process is unsuccessful
    3. Unfinish Redirect URL
      4. URL address where customer will be redirected when clicking "Back to Merchant Website" (Payment process cancelled).

When all of the above step is completed, VT-Web integration can be started.

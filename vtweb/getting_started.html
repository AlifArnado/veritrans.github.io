Getting Started

Get Client Library

Please download the VT-Web client library for either PHP, or Ruby to assist you on your integration process.

  • PHP
  • Ruby

Download the PHP library at our github page.

  1. Add these lines to your Gemfile:

    gem 'activemerchant', :git => 'https://github.com/veritrans/active_merchant.git'  
gem 'haml-rails', '~> 0.5.2'

    Don't forget to run "bundle install" from your command line.

  2. In the folder /config/initializers/, create new file 'active_merchant.rb':

    require 'active_merchant'  
require 'active_merchant/billing/integrations/action_view_helper'  
ActionView::Base.send(:include, ActiveMerchant::Billing::Integrations::ActionViewHelper)  
ActionView::Base.send(:include, ActiveMerchant::Billing::Integrations::Veritrans)  
ActiveMerchant::Billing::Base.integration_mode = :production
  3. Restart your server.

Development Configuration

You will need to know your merchant_id and merchant_hash_key to communicate with VT-Web. You can find the keys here. Please use the Development Configuration while you are still in development. Please switch to the Production Configuration only when you have received the bank approval and you are ready to go live.

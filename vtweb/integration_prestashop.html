VT-Web Prestashop Integration Steps

  1. Plugin Installation
  2. Transaction Test


  • Step 1Plugin Installation
  • Step 2Transaction Test

Requirements

  1. An online store with PrestaShop infrastructure. This plugin is tested with PrestaShop version 1.4, 1.5, and 1.6.
  2. PHP minimum version 5.3.
  3. Veritrans plugin for PrestaShop [ Github | Zip ].

Installation Process

  1. Extract the plugin you have previously downloaded and rename folder as veritranspay. Then Zip the folder back into veritranspay.zip.
  2. Access your PrestaShop administrator page.
  3. Go to menu Modules-modules.
  4. Choose Add a new module. Upload plugin file veritranspay.zip.
  5. Find the previously uploaded Veritrans module and click install.

Plugin Configuration

In order to configure Veritrans-PrestaShop plug-in:

  1. Access your PrestaShop admin page.
  2. Go to Modules-modules menu in PrestaShop admin page.
  3. In Veritrans module, click configure to enter plugin page.

  4. Insert required fields following the figure below.

  5. Login to MAP portal to configure a few options below:

    URL Role Redirect URL
    Payment Notification URL http://[your_shop’s_homepage]/index.php?fc=module&module=veritranspay&controller=notification
    Finish Redirect URL http://[your_shop’s_homepage]/index.php?fc=module&module=veritranspay&controller=confirmation
    Error Redirect URL http://[your_shop’s_homepage]/index.php?fc=module&module=veritranspay&controller=confirmation
    Unfinish Redirect URL http://[your_shop’s_homepage]/index.php

  1. Perform successful transaction in your online store by inputing a dummy credit card number as followed (Sandbox Mode):

    Type Visa
    Card Number 4811 1111 1111 1114
    CVV 123
    Exp. Month 01
    Exp. Year 2020

  2. Examine a few points below to ensure plugin is installed and performs properly.

    Check Point Expected Result When Unexpected Result Occurs
    Check order status in PrestaShop back end Order status turns into payment accepted Check endpoint/payment notification URL setting in MAP
    Merchant receives email notification Receive notification Check MAP menu setting - email notification
    Customer receives email notification Receive notification Check MAP menu setting - email notification

    NOTE:
    If the notifications not working well, please use the SEO URL feature from Prestashop.

    • Click on the preferences menu and choose SEO & URL.
    • On the SET SHOP URL submenu, fill with your domain configuration.

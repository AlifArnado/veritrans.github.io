Sample Code

Sample code in this documentation can be found at https://github.com/veritrans/papi-sample-code under folder VT-Web/PHP.

  1. Download all files under folder VT-Web/PHP. Placed it on to the server (For example: htdocs folder, if you are using XAMPP or MAMP).

  2. Change server_key under config.php according to value in Merchant Administration Portal (MAP) Setting » Access Keys.

  3. Open checkout.html in browser.

Activating 3DSecure

3DSecure transaction is activated by setting parameter ”credit_card_3d_secure” : true. The updated JSON format:

        "payment_type": "vtweb",
        "vtweb": {
            "enabled_payments": ["credit_card"],
            "credit_card_3d_secure":true
        },
        "transaction_details": {
            "order_id": "order45",
            "gross_amount": 200000

        "item_details": [{
            "id": "item1",

Handling Notification

notification_handler.php contains sample code for handling HTTP(S) POST notification. Configured the endpoint for merchant’s server in MAP under Setting » VT-Web Preferences.

Activating production environment:

Once you are finished with testing and ready to “go live”, make sure this :

  • Update the $endpoint in file checkout_process.php to: https://api.veritrans.co.id/v2/charge.
  • Change Server & Client key from sandbox MAP to Server & Client key from Production MAP.

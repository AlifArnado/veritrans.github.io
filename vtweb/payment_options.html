payment Options

Currently Veritrans has other payment option, namely :

  1. Mandiri ClickPay
  2. CIMB Clicks

Here is a tutorial for doing integration using the features to suit your needs.

Mandiri ClickPay

Mandiri Clickpay is a feature from Mandiri debit, where customer is able to proceed the transaction in merchant’s online shop with filling 16 digit of Mandiri Debit Card’s number then validating the transaction using PIN Mandiri Token

To activate Mandiri Clickpay, merchants needs to inform Veritrans sales representative about using Mandiri Clickpay for a payment method. If it's registered and merchant needs to add a Mandiri Clickpay as another payment method, parameters mandiri_clickpay should be added in the part of enable_payments so that JSON message will be.

  {
      "payment_type": "vtweb",
      "vtweb": {
          "enabled_payments": ["credit_card", "mandiri_clickpay"],
      },
      "transaction_details": {
          "order_id": "order45",
          "gross_amount": 200000
      },
  }

A new tab will appear on the VT-Web payment page, which shows you how to make a payment using Mandiri Clickpay.

CIMB Clicks

CIMB Clicks is a payment method using internet Banking from CIMB Niaga. Using CIMB Clicks, shoppers will be directed to CIMB Clicks payment page in order to proceed payment. For more information, please visit CIMB Niaga website.

To activate CIMB Clicks payment method, merchants needs to inform Veritrans sales representative about using CIMB Clicks for a payment method. If it's registered and merchant needs to add CIMB Clicks as another payment method, parameters cimb_clicks needs to be added in the part of enable_payments, so that JSON message will be.

  {
      "payment_type": "vtweb",
      "vtweb": {
          "enabled_payments": ["credit_card", "mandiri_clickpay", "cimb_clicks"],
      },
      "transaction_details": {
          "order_id": "order45",
          "gross_amount": 200000
      },
  }

A new tab will appear on the VT-Web payment page, which shows you how to make a payment using CIMB Clicks.

VT-Web Magento Integration Steps

  1. Plugin Installation
  2. Transaction Test


  • Step 1Plugin Installation
  • Step 2Transaction Test

Requirements

  1. An online store with Magento infrastructure. This plugin is tested with Magento version 1.9.
  2. PHP minimum version 5.3.
  3. Veritrans plugin for Magento [ Github | Zip ].

Installation Process

  1. Extract VTWebmagento-master.zip file you have previously downloaded.
  2. Upload / copy app, lib, dan skin folder that you have extracted into your Magento directory.

Plugin Configuration

In order to configure Veritrans-Magento plug-in:

  1. Access your Magento admin page.
  2. Go to System - Configuration menu in Magento admin page.
  3. In Sales-Payment Methods option, find Veritrans tab. If Veritrans tab is not present, click Save Config beforehand.
  4. Click general to start configuration.

  5. Insert required fields following the figure below and click Save Config.

  6. Login to MAP portal to configure a few options below:

    URL Role Redirect URL
    Payment Notification URL http://[your_shop’s_homepage]/vtweb/payment/notification
    Finish Redirect URL http://[your_shop’s_homepage]/
    Error Redirect URL http://[your_shop’s_homepage]/
    Unfinish Redirect URL http://[your_shop’s_homepage]/

  1. Perform successful transaction in your online store by inputing a dummy credit card number as follows (Sandbox Mode):
Type Visa
Card Number 4811 1111 1111 1114
CVV 123
Exp. Month 01
Exp. Year 2020
  1. Examine a few points below to ensure plugin is installed and performs properly.
Check Point Expected Result When Unexpected Result Occurs
Check order status in Magento back end Order status turns into processing Check endpoint/payment notification URL setting in MAP
Merchant receives email notification Receive notification Check MAP menu setting - email notification
Customer receives email notification Receive notification Check MAP menu setting - email notification

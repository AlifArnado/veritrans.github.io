VT-Web WooCommerce Integration Steps

  1. Plugin Installation
  2. Transaction Test


  • Step 1Plugin Installation
  • Step 2Transaction Test

Requirements

  1. An online store with WordPress and WooCommerce infrastructure. This plugin is tested with WordPress Version 3.9.1 and WooCommerce Version 2.1.11.
  2. PHP minimum version 5.3
  3. Veritrans plugin for WooCommerce [ Github | Zip ].

Installation Process

  1. Extract .zip file that you have previously downloaded.
  2. Upload the .zip extraction folder into WordPress installation directory wp-content/plugins/
  3. Activate Veritrans plugin from Plugin menu in your WordPress admin page.
  4. Insert http://[your web]/?wc-api=WC_Gateway_Veritrans link as Payment Notification URL in your MAP configuration.
  5. Insert http://[your web] link as Finish/Unfinish/Error Redirect URL in your MAP configuration.

Plugin Configuration

In order to configure Veritrans-WooCommerce plug-in:

  1. Access your WordPress admin page.
  2. Go to WooCommerce - Settings - Checkout menu in WordPress admin page.
  3. In Payment Gateway option, pick Veritrans then click settings.

  4. Insert required fields following the figure below.

VT-Web OpenCart Integration Steps

  1. Plugin Installation
  2. Transaction Test


  • Step 1Plugin Installation
  • Step 2Transaction Test

Requirements

  1. An online store with OpenCart infrastructure. This plugin is tested with OpenCart version 1.5.6.4.
  2. PHP minimum version 5.3.
  3. Veritrans plugin for OpenCart [ Github | Zip ].

Installation Process

  1. Extract vtweb-opencart-master.zip file you have previously downloaded.
  2. Upload admin, catalog, and system folder you extracted into your OpenCart directory.

Plugin Configuration

In order to configure Veritrans-OpenCart plug-in:

  1. Access your OpenCart admin page.
  2. Go to Extension - Payments menu in OpenCart admin page.
  3. Find and install Veritrans module to activate plugin.
  4. In Veritrans module, click Edit to enter plugin page.
  5. Insert required fields following the figure below.



  6. Click Save to save configuration.

  7. Login to MAP portal to configure a few options below:

    URL Role Redirect URL
    Payment Notification URL http://[your shop’s homepage]/index.php?route=payment/veritrans/payment_notification
    Finish Redirect URL http://[your shop’s homepage]/index.php?route=checkout/success&
    Error Redirect URL http://[your shop’s homepage]/index.php
    Unfinish Redirect URL http://[your shop’s homepage]/index.php
